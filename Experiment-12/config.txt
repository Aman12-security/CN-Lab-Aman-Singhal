# Commands Used in Experiment 12

1. **Create and edit a new script:**
   nano simple_script.sh

2. **Make the script executable:**
   chmod +x simple_script.sh

3. **Execute the script:**
   ./simple_script.sh

4. **Run a script with positional parameters:**
   ./param_script.sh param1 param2

5. **Arithmetic operations script:**
   ./arithmetic_script.sh

6. **Run a script with conditional statements:**
   ./conditional_script.sh

7. **Run a script with loops:**
   ./loop_script.sh
